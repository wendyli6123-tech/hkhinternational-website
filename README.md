## 🚀 Deployment Guide

### Step 1: Set up Formspree (Contact Form)

1. Go to [formspree.io](https://formspree.io) and sign up with your email
2. Create a new form, name it "HKH Inquiries"
3. Copy your form endpoint URL (looks like `https://formspree.io/f/xxxxxxx`)
4. Replace the URL in `index.html` line 322 with your actual endpoint

### Step 2: Deploy to Vercel (Recommended)

**Option A: Using GitHub**
1. Create a GitHub account at [github.com](https://github.com)
2. Create a new public repository named `hkh-hardware-website`
3. Push this code to your repository:
   ```bash
   git remote add origin https://github.com/your-username/hkh-hardware-website.git
   git push -u origin master
   ```
4. Go to [vercel.com](https://vercel.com), sign up with GitHub
5. Import your repository and click "Deploy"

**Option B: Direct Upload**
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop the `index.html` file directly - done in 30 seconds!
